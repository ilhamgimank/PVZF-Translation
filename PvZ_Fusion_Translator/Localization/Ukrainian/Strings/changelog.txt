<align=center><size=130%>Рослини проти Зомбі: Злиття</size></align>
<align=center><size=130%>Оновлення 3.4.2</size></align>

1. Нові рівні — "Я, Зомбі: Пригода":
 - Доступні через кнопку "Інша група рівнів" у режимі "Мозколомки".
 - Містять 3 розділи насичених рівнів і навчань із покроковими підказками.

2. Новий режим Одіссеї — Одіссея: Інвестиції:
 - Заснований на Одіссея: Чистилище. 
 - Зомбі, що з’являються хвилями, більше не дають балів, а безкоштовні покращення рослин прибрано. Натомість:
   - Ви починаєте з великої кількості очок, отримуєте нові після кожного раунду та можете відкривати Крамницю Одіссеї кожен раунд;
   - Епічні покращення Одіссеї продаються зі знижкою. Зберіть достатньо, щоб розблокувати Зв'язки з додатковими бонусами;
 - Нові Інвестиційні покращення: 
   - Обираються один раз на локацію. 
   - Вибір покращень Золотого або Призматичного рівня підвищує базову шкоду зв'язків, але збільшує шанс появи зомбі високого рівня та надає їм додаткове зменшення отримуваної шкоди.

3. Переробка режиму Одіссеї:
 - Крамниця Одіссеї оновлюється: 4 вищі розблокування Одіссеї + 4 випадкові покращення (було 3+2);
 - Покращено вибір випадкових покращень після раунду: якщо на полі є певний тип звичайної рослини Одіссеї, її покращення з’являються частіше (шанс у крамниці без змін);
 - Скасовано вимогу обирати 10 звичайних рослин Одіссеї на старті — тепер вони відкриті за замовчуванням;
 - Прокляття Чистилища та Прокляття Річкового Чистилища тепер мають окремі входи та комірки збереження;
 - Оптимізовано формат збережень Одіссеї (дані покращень зі старих збережень більше не сумісні);
 - Додано окремі смуги здоров’я для всіх міні-босів (можна увімкнути в налаштуваннях з головного меню);
 - Покращено порядок сортування звичайних покращень Одіссеї.

4. Нові параметри режиму Лотереї:
 - Доступні в налаштуваннях в головному меню;
 - Можна вмикати/вимикати: Епічних зомбі Одіссеї, Мінібосів та Еволюційних зомбі;
 - Якщо ввімкнені всі три, рослини Одіссеї мають 10% шанс стати своїми Зоряними формами.

5. Нові рослини:
 - Базова рослина:
  - Перцеверк;
 - Базові злиття:
  - Відскок-гриб (Димо-гриб + Капусто-пульта);
  - Тиснявка (Споро-гриб + Чавидло);
  - Фатальний Гарбуз (Фаталь-гриб → Гарбуз);
  - Шипований горщик (Шипотрав + Квітковий горщик);
  - Ламповий Горохостріл (Горохостріл + Ліхтарник);
  - Чавидло-дрон (Чавидло + Вентилист);
  - Озима Капусто-пульта (Льодо-гриб + Капусто-пульта);
  - Озима Кукурудзо-пульта (Льодо-гриб + Кукурудзо-пульта);
  - Плуторостик (Споро-гриб → Плуторості);
  - Капустяна Чавидло-пульта (Чавидло + Капусто-пульта);
 - Злиття Одіссеї:
  - Звичайні:
   - Парасольник Скаді (Парасольник Фароса + Льодо-гриб);
   - Квазар (Механічного Гарбузa + Зорефрукт);
   - Маслопостний горщик (Маслоконвеєр + Квітковий горщик);
   - Бамбракон (Бамбракон + Піробах);
  - Вищі:
   - Пеклокриг (Гострольодик + Гострольодик).

6. Нові зомбі:
 - Звичайні:
  - Футболіст-Хамелеон;
  - Проривний Футболіст;
 - Міні-боси:
  - Нетлінний дракон
  - Загін Альфа-Асасинів тепер стає Загоном Омега-Асасинів, якщо з’являється як міні-бос.

7. Нові костюми:
 - Грибогедон → Цвітогедон.

8. Нове маля:
 - Придбайте нове покращення, щоб розблокувати Маля Джилл, яка битиметься разом із вами під час забігів Одіссеї.

9. Технічні покращення:
 - Додано опцію \"Пропустити вибір рослин\" у Безкінечних режимах;
 - Додано гарячу клавішу B для відкриття/закриття екрана вибору рослин (за наявності);
 - Додано гарячу клавішу H для для перегляду характеристик рослини в позиції курсора;
 - Додано гарячу клавішу F10 для приховування металевих об'єктів;
 - Оптимізовано внутрішній код для зменшення лагів, особливо якщо вимкнено відображення здоров’я зомбі.

10. Баланс:
 - Гіга-Мехаріх: Тепер може відновлювати максимум 10 млн здоров’я за раунд;
 - Пеклокриг: Максимальна к-сть на галявині — (к-сть смуг) + 2;
 - Механічний Гарбуз отримав свої власні покращення Одіссеї;
 - Мехаріх 3.О.М.10 отримав свої власні покращення Одіссеї.

 - Рослини:
  - Похмурий Арбітр → Фатальний Суддя;
  - Вибухомет → Вибух-О-мет;
  - Зимовий кавун → Озимий кавун;
  - Пружній кавун → Весняний кавун;
  - Айсберго-пульта → Озима капуста;
  - Чавидло-пульта → Кавунова Чавидло-пульта;
  - Вершково пульта → Масляна Чавидло-пульта;
 - Рівні:
  - День → Ліс;
  - Басейн → Річка;
  - Дах → Гори;
  - Сніг → Зима;
  - Обернений басейн → Місток;
  - Заморожена річка → Зимова річка;

12. Виправлення помилок:
 - Виправлено проблему, через яку деякі зомботані не могли бути оглушені маслом через некоректне визначення голови;
 - Виправлено низку інших помилок.
